Linux安装Vmware-tools错误解决

The path “” is not a valid path to the xxxx-generic kernel headers.

解决办法

装上重新运行安装文件，如果还不行，就要手动搞一下
去目录 /usr/src/linux-headers-xxxx/include/uapi/linux/dvb/
把version.h 复制到/usr/src/linux-headers-xxxx/include/linux下，重新vmware-install.pl 执行安装
